**=====Приложение для отслеживания расходов=====**

CLI-интерфейс 

Требования: 
1. Пользователи могут добавить расходы, указав описание и сумму
2. Пользователи могут обновлять информацию о расходах 
3. Пользователи могут удалить расходы
4. Пользователи могут просмотреть все расходы 
5. Пользователи могут просмотреть общую сводку расходов
6. Пользователи могут просмотреть сводку расходов за конкретный месяц 

Дополнительный функционал:
1. Категории расходов и фильтрация расходов по категориям
2. Установка бюджета пользователя на каждый месяц (При превышении бюджета выводить предупреждение)
3. Экспорт расходов в CSV-файл
4. У каждого пользователя есть своя УЗ (логин-пароль)
5. Логирование событий

**=====Структура проекта=====**

1. Структура пользователя (Имя, Фамилия)

**=====Структура расходов=====**
1. (Описание, сумма)✅
2. Добавить расход ✅
3. Удалить расход ✅
4. Просмотр списка расходов ✅
5. Общая сумма расходов ✅
6. Обновление информации о расходах✅ 
7. Сумма расходов за конкретный месяц 
8. Сводка по расходам за конкретный месяц
